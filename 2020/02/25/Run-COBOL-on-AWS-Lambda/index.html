Run COBOL on AWS Lambda

5 simple steps to deploy and run COBOL program on AWS Lambda.

  1. Make docker environment
    Install and test necessary docker apps on Amazon Linux.

  2. Make COBOL compiling environment
    Create Dockerfile to install cobol compiler, tools and build lambda function package.

    Create bootstrap script for AWS Lambda Custom Runtime, which will be used to lunch COBOL program.

    Prepare build.sh.

  3. Build
    Build the Zip file for AWS Lambda.

  4. Deploy
    Deploy the Zip file by appropriate IAM role.

  5. Run it
    Just use any test event to run your COBOL program on AWS Lambda.
